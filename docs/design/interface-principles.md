# Interface Principles

The app should feel like a live command center without sacrificing clarity.

- One main playback workspace.
- Dense but secondary intelligence panels.
- Always-visible peer, room, sync, and media status.
- Dark technical shell with restrained neon accents.
- Thin borders, compact controls, monospace labels, and minimal rounding.
- Motion only when it communicates state.

The visual rule:

```text
Make it look cinematic, but make it behave like a serious tool.
```
